3- A XML VIEW e a JS VIEW tem inicialmente a diferença de linguagem de escrita, visto que XML é uma linguagem 
de marcação e JS é uma linguagem de programação, mas acredito que a principal diferença entre as duas seria a
padronização de codigo, pois a XML view é o padrão ideal para o MVC, já a JS VIEW é feita via controller
instanciando os objetos via JS.
